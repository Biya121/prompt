레시피 패턴
구체적인 순서가 필요할 때 사용합니다. "수출 자동화 어떻게 해?" → 두루뭉술한 답. 레시피 패턴은 목표 → 현재 상태 → 단계별 절차를 순서 있는 행동 지침으로 만들어냅니다.
기본 레시피 패턴
#목표
Python 초보자가 무역 HS code 자동 분류 프로그램 만들기

#현재 상태
- Python 설치만 되어 있음
- 프로그래밍 경험 없음
- 엑셀로 품목 데이터 500개 보유

#요청
위의 목표 달성을 위한 단계별 레시피를 작성해 주세요.
각 단계는 아래 형식으로.

##STEP {번호} : {단계명}
- 할 일 : {구체적 행동}
- 소요 시간 : {예상 기간}
- 완료 기준 : {이것이 되면 다음으로}
- 주의사항 : {흔한 실수}
분기 상황 포함 레시피 패턴
ℹ️
각 단계에서 결과에 따라 다음 행동이 달라지는 분기를 함께 정의합니다.
무역 계약 클레임 절차를 단계별로 작성해 주세요.
단, 각 단계에서 발생할 수 있는 분기 상황도 포함.

형식 :
STEP N. {행동}
- 정상 : {이 결과면 다음 단계로}
- 이상 : {이 결과면 어떻게 대응}
