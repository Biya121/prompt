프롬프트의 세 가지 유형
AI에게 무엇을 주느냐에 따라 세 유형으로 나뉩니다. 목적에 맞는 유형을 선택하면 결과물의 품질이 크게 달라집니다.
유형 비교
유형핵심언제 쓰나
① 서술형맥락과 배경을 설명AI가 상황을 이해해야 할 때
② 지침형단계별 규칙을 제시일관된 형식의 결과가 필요할 때
③ 함수형입출력을 명확히 정의반복 사용하거나 자동화할 때
① 서술형 — 맥락을 준다
맥락과 배경을 설명하여 AI가 상황을 이해하도록 유도합니다. AI에게 상황을 충분히 설명할 때 사용합니다.
당신은 10년 경력의 무역 전문가입니다.
중소기업 수출 담당자가 HS code를 처음
배우는 상황입니다. 이 사람에게 HS code의
개념과 중요성을 설명해 주세요.
② 지침형 — 규칙을 준다
단계별 절차나 명확한 규칙을 제시하여 원하는 출력을 유도합니다. 일관된 형식의 결과가 필요할 때 사용합니다.
다음 규칙에 따라 Python 코드를 작성하세요.
[1] 함수명은 snake_case로 작성.
[2] 각 함수에 docstring을 포함.
[3] try-except 예외 처리 반드시 포함.
[4] 파일 반드시 json 형식으로 변환.
③ 함수형 — 구조를 준다
입력값과 출력값을 명확히 정의하여 일관된 결과를 생성합니다. 반복 사용하거나 자동화할 때 가장 강력합니다.
입력 : {product_name}, {hs_code}, {country_code}
처리 : 해당 국가의 관세율 및 수입 규제 조회
출력 :
{
  "관세율": "",
  "규제사항": [],
  "필요서류": []
}
✅
결과 예시
관세율: "기본 6.4% (한-미 FTA 적용 시 0% 무관세)"
규제사항: FDA 시설 등록, 사전 통보, 라벨링, FSMA 준수 등
필요서류: 상업송장, 포장명세서, 선하증권, FDA 통보 확인서 등
