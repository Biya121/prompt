인지 검증자 패턴
복잡한 질문을 바로 던지면 오류와 누락이 생깁니다. 인지 검증자 패턴은 AI가 먼저 하위 질문으로 분해하고, 각각을 검증한 뒤, 통합 판단을 내립니다.
일반 패턴 vs 인지 검증자
방식처리 흐름품질
일반 패턴복잡한 질문 → 즉각 대답오류·누락 가능성 높음
인지 검증자질문 분해 → 각 검증 → 통합 답변정확성과 깊이 높음
방법 1 · 명시적 분해 요청
아래 질문에 바로 답하지 마세요.
먼저 이 질문을 답하기 위해 필요한
하위 질문 목록을 만들고,
각 하위 질문에 순서대로 답한 뒤,
최종 통합 답변을 제시하세요.

질문 :
우리 회사가 베트남에 생산 법인 설립 시
세금, 인력, 물류 측면에서 유리한가?
방법 2 · 전문가 검증 방식
{전문 분야} 전문가처럼 이 문제를 다루세요.

- 확신이 없는 부분은 "가정"으로 명시하세요.
- 각 판단의 근거를 함께 제시하세요.
- 단계적으로 결론에 도달하세요.
