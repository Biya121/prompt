메타언어 생성 패턴
반복적으로 사용하는 지시를 짧은 명령어(단축어)로 미리 정의해 두는 패턴입니다. 매번 긴 지시를 입력하는 대신, 약속된 단축어 하나로 복잡한 작업을 즉시 실행할 수 있습니다. 자주 쓰는 작업 방식이 있다면 나만의 명령어 체계를 만들어 보세요.
기본 구조
🔑
단축어를 먼저 정의 → AI가 기억 → 단축어만 입력하면 해당 작업 즉시 실행
예시 1 · 무역 실무용 단축어 세트
자주 사용하는 출력 형식(요약, 코드, 비교)을 단축어로 미리 정의합니다. 한 번만 설정해 두면 이후 대화에서 짧은 명령어로 원하는 형식의 답변을 바로 받을 수 있습니다.
이 대화에서 아래 명령어를 사용합니다.
각 명령어의 정의를 정확하게 기억하고 실행하세요.

/요약 : 핵심만 3줄, 전문 용어 제거,
        무역 실무자 기준, 결론 먼저 → 이유 순서로 작성

/코드 : Python 작성, 주석 한국어, 예외 처리 포함,
        실행 가능한 완성 코드, 설명 없이 코드만 출력

/비교 : 2~4개 대안을 표로 정리,
        항목 : 장점/단점/비용/난이도/추천 상황,
        마지막 행에 종합 추천 추가

명령어를 이해했다면 "준비 완료."라고만 답하세요.
✅
사용 예시
설정 후 → /요약 베트남 FTA 활용 방법 알려 줘.
AI가 자동으로 3줄 요약, 결론 우선, 실무자 언어로 답변합니다.
예시 2 · 교정 작업용 단축어 세트
특정 역할(편집자, 교사 등)과 대상(독자 수준)을 정의한 뒤, 단축어로 작업 종류를 구분합니다. 파라미터를 함께 넘길 수도 있습니다.
#입력
[독자] = 초등학생

[단축어1] = "/유:[단어]:[개수]"
→ [단어]에 대한 [독자]가 이해할 수 있는
  유의어 [개수]개를 제공하고,
  가장 추천하는 유의어를 알려 주세요.

[단축어2] = "/반:[단어]:[개수]"
→ [단어]에 대한 [독자]가 이해할 수 있는
  반의어 [개수]개를 제공하고,
  가장 추천하는 반의어를 알려 주세요.

[단축어3] = "/유반:[단어]:[개수]"
→ [단축어1]과 [단축어2]를 동시에 수행하세요.

#처리
당신은 "출판사 편집자"입니다.
[독자] 대상의 교양 도서를 교정하고 있습니다.
위 내용을 기억하고, 단축어를 받으면 즉시 실행하세요.
단축어를 받을 때까지 대기합니다.
✅
사용 예시
/유:행복:3 → 행복의 유의어 3개 (기쁨, 즐거움, 만족) + 가장 추천 : 기쁨
/유반:슬픔:2 → 유의어 2개 + 반의어 2개 동시 출력
나만의 단축어 설계 팁
설계 요소고려 사항예시
명령어 기호일반 문장과 구분되는 기호 사용/, !, @
파라미터 구분입력값을 전달할 때 구분자 지정/유:[단어]:[개수]
기억 확인설정 후 AI에게 이해 여부 확인"이해했다면 '준비 완료.'라고 답하세요."
범위 제한이 대화에서만 유효함을 명시"이 대화에서 아래 명령어를 사용합니다."
