다중 관점 기법
하나의 주제를 단일 시각으로만 바라보면 놓치는 것이 생깁니다. 다중 관점 기법은 서로 다른 입장, 전문 분야, 시간대를 동시에 소환해 문제를 입체적으로 분석합니다. 관점들이 충돌하는 지점에서 진짜 문제가 보이고, 동의하는 지점에서 확실한 진실이 드러납니다.

관점강점빠지는 것
전문가 관점만실용성 높음감성·공감 부재
사용자 관점만공감 높음전문성 부재
낙관론만가능성 포착리스크 간과

다중 관점이 만드는 것
→
충돌 지점 서로 다른 관점이 부딪히는 곳에 진짜 문제가 숨어 있습니다.
→
동의 지점 모든 관점이 동의하는 곳이 확실한 진실입니다.
→
보완 지점 관점들이 서로를 채워주는 곳에 완전한 해결책이 있습니다.

시간과거의 결정이 현재에 어떤 영향을 주고, 지금의 선택은 미래를 어떻게 바꾸는가?
가치관어떤 철학과 우선순위로 보느냐에 따라 답이 달라지는가?
문화·맥락다른 환경과 배경에서는 이 문제가 어떻게 보이는가?
방법 1 · 병렬 관점 소환
여러 관점을 동시에 독립적으로 분석한 뒤, 마지막에 교차점을 찾습니다. 단순 나열이 아니라 관점 간의 교차점을 찾는 것이 핵심입니다.
아래 주제를 4가지 관점에서 동시에 분석하세요.
각 관점은 독립적으로 분석하되,
마지막에 관점들이 만나는 교차점을 찾아 주세요.

관점 : 경제학 / 심리학 / 법률 / 현장 실무자

주제 : "중소기업 수출 자동화 도입의 득과 실"
방법 2 · 대화 시뮬레이션
서로 다른 입장의 인물들이 한 테이블에 앉아 토론하는 장면을 AI가 연출합니다. 각자의 주장과 반론, 그리고 결론까지 이어지는 구조입니다. 이해관계자 갈등을 미리 예측하거나 설득 논리를 준비할 때 유용합니다.
아래 상황에 관련된 4명의 이해관계자를 동시에 분석하세요.

각 이해관계자에 대해 :
- 가장 원하는 것 (핵심 욕구)
- 가장 두려워하는 것 (핵심 공포)
- 공개적으로 말하는 것 vs 속으로 생각하는 것
- 이 상황에서 갖는 협상력

마지막에 :
- 모두가 동의할 수 있는 최소 공통점
- 가장 큰 갈등 지점
- 현실적 합의 가능 경로

상황 : 중소기업 수출 자동화 시스템 도입을 추진 중인 상황.
이해관계자 : 대표 / 무역 팀장 / 팀원 2명 / IT 외주업체
방법 3 · 전문 분야 렌즈 분석
같은 문제를 전혀 다른 전문 분야의 시각으로 바라봅니다. 각 분야가 문제를 어떻게 정의하고 어떤 해결책을 제안하는지 비교하면, 예상치 못한 해결 실마리가 나타납니다.
아래 문제를 5가지 전문 분야의 렌즈로 동시에 분석하세요.
각 분야가 이 문제를 어떻게 정의하고
어떤 해결책을 제안하는지 보여 주세요.

분야 : 심리학 / 경제학 / 시스템 공학 / 인류학 / 게임 디자인

문제 : "무역팀이 새로운 자동화 도구 도입 후
       3개월이 지나도 제대로 활용하지 못한다."
방법 4 · 시간 레이어 분석
과거→현재→미래를 연결된 레이어로 분석합니다. 과거의 결정이 현재에 어떻게 나타나고, 지금의 선택이 미래를 어떻게 바꾸는지를 흐름으로 보여줍니다. 구조적 문제나 장기 전략 수립에 적합합니다.
아래 주제를 3개의 시간 레이어에서 동시에 분석하세요.
각 레이어가 다음 레이어에 어떤 영향을 주는지 연결해 주세요.

레이어 1 — 과거 (5년 전)
당시 사람들은 무엇을 알고, 무엇을 몰랐나?

레이어 2 — 현재
과거의 결정이 현재에 어떤 모습으로 나타나는가?
지금 진짜 문제는 무엇인가?

레이어 3 — 미래 (5년 후)
지금의 선택이 미래에 어떤 모습으로 누적되는가?
어떤 결정을 내려야 미래가 바뀔 수 있는가?

주제 : "한국 중소기업의 무역 디지털 격차"
💡
언제 쓰면 좋은가?
단순 정보 조회보다 의사결정, 갈등 분석, 전략 수립처럼 복잡한 상황에서 진가를 발휘합니다. 관점의 수는 3~5개가 적당하며, 너무 많으면 분석이 산만해집니다.
